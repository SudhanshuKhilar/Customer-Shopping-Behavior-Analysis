📊 Customer Behavior & Revenue Analysis

📌 Project Overview
This project presents a complete end-to-end Data Analytics workflow focused on analyzing retail customer shopping behavior. The objective is to transform raw transactional data into actionable business insights that improve customer engagement, revenue optimization, and strategic decision-making.
Using a dataset of customer transactions across multiple product categories, this project explores purchasing patterns, customer segmentation, subscription impact, discount usage, and revenue contribution across demographics.
The project simulates a real-world business scenario where management wants to understand:
What drives customer purchase decisions?
Which customer segments generate the highest revenue?
How do discounts and subscriptions impact profitability?
Which products and categories perform best?

🛠️ Tools & Technologies Used
Python (Pandas, NumPy, Matplotlib, Seaborn) – Data cleaning, preprocessing, feature engineering
SQL (SQL Server) – Business queries, segmentation analysis, revenue insights
Power BI – Interactive dashboard and data visualization
GitHub – Project documentation and version control

🔍 Project Workflow

1️⃣ Data Preparation & Modeling (Python)
Data loading and structure validation
Handling missing values (Review Rating imputation)
Column standardization
Feature engineering (age groups, purchase frequency)
Data consistency checks
Database integration with SQL Server

2️⃣ Business Analysis (SQL)

Revenue comparison by gender
High-spending discount users identification
Top-rated products analysis
Shipping type revenue comparison
Subscription vs non-subscription impact
Customer segmentation (New, Returning, Loyal)
Revenue contribution by age group

3️⃣ Dashboard & Visualization (Power BI)
An interactive dashboard was built to present:
Revenue trends
Customer segments
Subscription performance
Product performance
Shipping behavior insights

📈 Key Insights

Subscription customers show higher spending and loyalty rates
Express shipping users spend more per transaction
Certain products are highly discount-dependent
Loyal customers contribute a significant share of revenue
Targeted marketing by age group can increase ROI

💡 Business Recommendations

Strengthen subscription-based loyalty programs
Optimize discount strategy to balance sales and margins
Promote top-rated products in marketing campaigns
Focus retention efforts on converting new customers into loyal customers
Leverage demographic-based targeting for personalized campaigns
